# Bruuhim's Seanime Extensions

A collection of manga providers and other extensions for [Seanime](https://github.com/5rahim/seanime).

| **LekManga** | Arabic   | `https://raw.githubusercontent.com/bruuhim/bruuhim-seanime-extensions/main/manga-providers/lekmanga/lekmanga.json` |

## Other Extensions

- **MAL-Button-Seanime**: Add a Seanime button to MyAnimeList.
- **Open-in-Seanime**: Quick access to Seanime from your browser.

---

### How to Install

1. Copy the **Installation URL** for the extension you want.
2. Open Seanime and navigate to **Settings > Extensions**.
3. Paste the URL into the **External Manifest URL** field and click Install.
